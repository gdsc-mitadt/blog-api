# GDSC-ADTU BLOG API

* pip install -r requirements.txt

### dot-env Structure
DEBUG=False<br>
DB_URI=""<br>
DB_NAME=""<br>
JWT_SECRET_KEY=""

### Run Command
* Method one :
1. export FLASK_APP=run.py
2. export FLASK_ENV=development
3. flask run

* Method two :
1. Launch your bash terminal, and type
2. ./run
