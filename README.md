# GDSC BLOG API

### Requirements
* pip install -r requirements.txt

### dot-env Structure
DEBUG=False
DB_URI=""
DB_NAME=""
JWT_SECRET_KEY=""

### Run Command
* Method one :
1. export FLASK_APP=run.py
2. export FLASK_ENV=development
3. flask run

* Method two :
1. Launch your bash terminal, and type
2. ./run
